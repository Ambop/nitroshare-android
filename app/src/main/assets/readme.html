Thank you for helping test NitroShare for Android. This guide provides useful information for using the application.

Usage

  • To send a transfer, use the share button in another application and select the "Send with NitroShare" item
  • To receive a transfer, simply initiate one from another device, such as a PC
  • To adjust settings, such as this device's name on the network, use the settings menu item in the upper-right corner

Limitations

The application is still in its early stages. As such, certain features that will be included in the final release are absent or incomplete in the beta builds.

  • It is not possible to send a directory due to the way third-party applications share data with NitroShare. A simple file manager will be added to NitroShare to work around this limitation.

Reporting Bugs

To report a bug, please visit this page. If you do not wish to create a GitHub account, you can send an email to nathan@nitroshare.net instead.
